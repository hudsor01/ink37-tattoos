Query pattern mapping (from 08-RESEARCH.md):
| Prisma | Drizzle |
|--------|---------|
| `db.model.findMany({ where, include, orderBy, skip, take })` | `db.query.model.findMany({ where: eq(...), with: {...}, orderBy: [desc(...)], offset, limit })` |
| `db.model.findUnique({ where: { id } })` | `db.query.model.findFirst({ where: eq(model.id, id) })` |
| `db.model.create({ data })` | `db.insert(model).values(data).returning()` |
| `db.model.update({ where, data })` | `db.update(model).set(data).where(eq(...)).returning()` |
| `db.model.delete({ where })` | `db.delete(model).where(eq(...)).returning()` |
| `db.model.upsert(...)` | `db.insert(model).values(data).onConflictDoUpdate({ target, set })` |
| `db.model.count({ where })` | `db.select({ count: sql\`cast(count(*) as integer)\` }).from(model).where(...)` |
| `db.model.aggregate({ _sum })` | `db.select({ total: sql\`sum(${model.col})\` }).from(model)` |
| `db.model.groupBy(...)` | `db.select({...}).from(model).groupBy(model.col)` |
| `db.$transaction(async tx => {...})` | `db.transaction(async (tx) => {...})` |
| `db.$transaction([p1, p2])` | `db.transaction(async (tx) => { await tx...; await tx...; })` |
| `{ amount: { increment: val } }` | `set({ amount: sql\`${model.amount} + ${val}\` })` |
| `include: { rel: { select: {...} } }` | `with: { rel: { columns: {...} } }` |
| `where: { OR: [{...}, {...}] }` | `where: or(eq(...), eq(...))` |
| `where: { name: { contains: x, mode: 'insensitive' } }` | `where: ilike(model.name, \`%${x}%\`)` |
| `Prisma.InputJsonValue` | `Record<string, unknown> \| null` |
| `Prisma.DbNull` | `null` |
| `include: { _count: { select: { items: true } } }` | Subquery or separate count |
| Prisma error P2025 | Check `result === undefined` (Drizzle returns undefined, not throw) |

Type inference:

type NewCustomer = typeof schema.customer.$inferInsert;
