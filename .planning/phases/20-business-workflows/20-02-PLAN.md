       - **Per research pitfall 4:** Calculate true balance from payment table, NOT from session.paidAmount. Query:
         ```
         SELECT ts.id, ts."totalCost", ts."designDescription", ts."customerId",
                COALESCE(SUM(CASE WHEN p.status = 'COMPLETED' THEN p.amount ELSE 0 END), 0) as total_paid
         FROM tattoo_session ts
         LEFT JOIN payment p ON p."tattooSessionId" = ts.id
         WHERE ts.status = 'SCHEDULED'
         GROUP BY ts.id
         HAVING ts."totalCost" > COALESCE(SUM(CASE WHEN p.status = 'COMPLETED' THEN p.amount ELSE 0 END), 0)
         ```
         Use Drizzle SQL builder with `sql` template literals for this aggregation (relational API does not support aggregations per CLAUDE.md).
