**src/lib/dal/products.ts:**
```typescript
import 'server-only';
import { cache } from 'react';
import { db } from '@/lib/db';
import { getCurrentSession } from '@/lib/auth';
import { redirect } from 'next/navigation';

const STAFF_ROLES = ['staff', 'manager', 'admin', 'super_admin'];

async function requireStaffRole() {
  const session = await getCurrentSession();
  if (!session?.user) redirect('/login');
  if (!STAFF_ROLES.includes(session.user.role)) {
    throw new Error('Insufficient permissions: requires staff role or above');
  }
  return session;
}
```

Exports:
- `getActiveProducts` (public, no auth): `db.product.findMany({ where: { isActive: true }, orderBy: [{ sortOrder: 'asc' }, { createdAt: 'desc' }] })` -- used by store catalog page. Cache-wrapped.
- `getProductById` (public, no auth): `db.product.findUnique({ where: { id } })` -- used by product detail page. Cache-wrapped.
- `getProducts` (admin, requireStaffRole): `db.product.findMany({ orderBy: [{ sortOrder: 'asc' }, { createdAt: 'desc' }], include: { _count: { select: { orderItems: true } } } })` -- used by admin product list. Cache-wrapped.
- `createProduct`: requireStaffRole, accepts `CreateProductData` (from validation.ts), creates db record, returns product.
- `updateProduct`: requireStaffRole, accepts `UpdateProductData`, updates db record, returns product.
- `deleteProduct`: requireStaffRole, accepts product id string, soft-deletes by setting `isActive: false`.

**1. src/lib/actions/product-actions.ts** -- Admin product CRUD Server Actions:
```typescript
'use server';
```
Following exact pattern from `payment-actions.ts`:
- `createProductAction(formData: FormData)`: getCurrentSession auth check, parse with `CreateProductSchema`, call Stripe `stripe.products.create` and `stripe.prices.create` to sync (per D-01), call `createProduct` DAL with stripeProductId and stripePriceId, logAudit fire-and-forget, revalidatePath('/dashboard/products'), return `{ success: true, product }`.
- `updateProductAction(formData: FormData)`: getCurrentSession, parse with `UpdateProductSchema`, if price changed: create new Stripe Price (`stripe.prices.create`), archive old Price (`stripe.prices.update({ active: false })`), update stripePriceId. If name/description changed: `stripe.products.update`. Call `updateProduct` DAL, logAudit, revalidatePath, return `{ success: true, product }`.
- `deleteProductAction(formData: FormData)`: getCurrentSession, get productId from FormData, call `deleteProduct` DAL (soft delete), archive Stripe Product if stripeProductId exists (`stripe.products.update({ active: false })`), logAudit, revalidatePath, return `{ success: true }`.
