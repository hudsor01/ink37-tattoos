    - Add `productImage` table after the `product` table definition:
      ```
      id: uuid defaultRandom primaryKey
      productId: uuid notNull references product.id onDelete cascade
      url: text notNull
      alt: text
      sortOrder: integer notNull default(0)
      isVisible: boolean notNull default(true)
      createdAt: timestamp notNull defaultNow
      ```
    - Add indexes: `product_image_productId_idx` on productId, `product_image_productId_sortOrder_idx` on (productId, sortOrder)
    - Add `productImageRelations`: belongsTo product
    - Update `productRelations`: add `images: many(productImage)`
    - Add to `order` table: `trackingNumber: text('trackingNumber')`, `trackingCarrier: text('trackingCarrier')`

    **3. Create SQL migration** at `src/lib/db/migrations/add-product-images-and-order-tracking.sql`:
    ```sql
    CREATE TABLE IF NOT EXISTS "product_image" (
      "id" uuid PRIMARY KEY DEFAULT gen_random_uuid(),
      "productId" uuid NOT NULL REFERENCES "product"("id") ON DELETE CASCADE,
      "url" text NOT NULL,
      "alt" text,
      "sortOrder" integer NOT NULL DEFAULT 0,
      "isVisible" boolean NOT NULL DEFAULT true,
      "createdAt" timestamp NOT NULL DEFAULT now()
    );
    CREATE INDEX IF NOT EXISTS "product_image_productId_idx" ON "product_image" ("productId");
    CREATE INDEX IF NOT EXISTS "product_image_productId_sortOrder_idx" ON "product_image" ("productId", "sortOrder");
    ALTER TABLE "order" ADD COLUMN IF NOT EXISTS "trackingNumber" text;
    ALTER TABLE "order" ADD COLUMN IF NOT EXISTS "trackingCarrier" text;
    ```

    **4. Create product image DAL** at `src/lib/dal/product-images.ts`:
    - `import 'server-only'` at top
    - `getProductImages(productId: string)`: cache-wrapped, requireStaffRole, query productImage where productId, order by sortOrder asc
    - `createProductImage(data: { productId: string; url: string; alt?: string })`: requireStaffRole, get max sortOrder for product, insert with sortOrder = max + 1, return the new image
    - `updateImageVisibility(id: string, isVisible: boolean)`: requireStaffRole, update productImage set isVisible where id, returning
    - `reorderProductImages(productId: string, orderedIds: string[])`: requireStaffRole, wrap in transaction, for each id in orderedIds update sortOrder to its index
    - `deleteProductImage(id: string)`: requireStaffRole, delete from productImage where id, returning

    **5. Create product image actions** at `src/lib/actions/product-image-actions.ts`:
    - `addProductImageAction(productId: string, url: string, alt?: string)`: safeAction wrapper, call createProductImage, audit log, revalidatePath(`/dashboard/products/${productId}/edit`), return ActionResult
    - `toggleImageVisibilityAction(imageId: string, isVisible: boolean, productId: string)`: safeAction wrapper, call updateImageVisibility, audit log, revalidatePath, return ActionResult
    - `reorderProductImagesAction(productId: string, orderedIds: string[])`: safeAction wrapper, validate all IDs are UUIDs, call reorderProductImages, audit log, revalidatePath, return ActionResult
    - `deleteProductImageAction(imageId: string, productId: string)`: safeAction wrapper, call deleteProductImage, audit log with Vercel Blob cleanup note, revalidatePath, return ActionResult

    - Props: `images: ProductImage[]`, `onReorder: (orderedIds: string[]) => Promise<void>`, `onToggleVisibility: (id: string, visible: boolean) => Promise<void>`, `onDelete: (id: string) => Promise<void>`, `onAdd: (url: string) => Promise<void>`
    - Each image item is a SortableItem using useSortable hook with CSS.Transform.toString for style

    - Dimmed opacity on hidden images (opacity-50 + "Hidden" badge)
    - On drag end: compute new order from arrayMove, call onReorder with new ID order
