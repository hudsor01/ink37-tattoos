**Part A: Rewrite notifications-cleanup route (CRON-SEC-01, CRON-SEC-02, CRON-CLEAN-01)**

Rewrite `src/app/api/cron/notifications-cleanup/route.ts`:

1. Remove the `import { Redis } from '@upstash/redis'` line.
2. Add `import { verifyCronAuth, getRedis } from '@/lib/security/cron-auth';`
3. Add `import crypto from 'node:crypto';`
4. Add `import { env } from '@/lib/env';`

5. Replace the inline `tryAcquireLock()` function. New version:
   - Gets Redis via `getRedis()` (not `new Redis()`)
   - If null (no Redis configured), log warning and return `{ acquired: true, owner: null }`
   - Generates `const owner = crypto.randomUUID()`
   - Calls `redis.set(LOCK_KEY, owner, { nx: true, ex: LOCK_TTL_SECONDS })`
   - If result === 'OK', returns `{ acquired: true, owner }`
   - Otherwise returns `{ acquired: false, owner: null }`
   - Return type: `Promise<{ acquired: boolean; owner: string | null }>`

6. Replace the inline `releaseLock()` function. New version:
   - Signature: `async function releaseLock(owner: string | null): Promise<void>`
   - If owner is null, return early
   - Gets Redis via `getRedis()` (same singleton as tryAcquireLock)
   - If null, return early
   - Uses Lua script for atomic check-and-delete:
     ```typescript
     const RELEASE_SCRIPT = `
       if redis.call("get", KEYS[1]) == ARGV[1] then
         return redis.call("del", KEYS[1])
       else
         return 0
       end
     `;
     await redis.eval(RELEASE_SCRIPT, [LOCK_KEY], [owner]);
     ```

7. Replace the auth block in POST handler (lines 66-81). New version:
   ```typescript
   const auth = verifyCronAuth(request);
   if (!auth.valid) {
     return NextResponse.json({ error: auth.error }, { status: auth.status });
   }
   ```

8. Replace the parseInt calls (lines 95-97). Since env() now returns coerced numbers:
   ```typescript
   const readRetentionDays = env().NOTIFICATION_RETENTION_READ_DAYS ?? 30;
   const unreadRetentionDays = env().NOTIFICATION_RETENTION_UNREAD_DAYS ?? 90;
   const batchSize = env().NOTIFICATION_CLEANUP_BATCH_SIZE ?? 1000;
   ```

9. Update the lock usage in the handler:
   ```typescript
   const { acquired, owner } = await tryAcquireLock();
   if (!acquired) { ... }
   // ... in finally block:
   await releaseLock(owner);
   ```
